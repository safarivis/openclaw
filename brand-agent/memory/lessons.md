# Lessons Learned

Insights from brand successes. and failures to apply in future work.

---

## How to Use This File

### When to Add Lessons
- After a campaign underperforms or exceeds expectations
- When brand feedback reveals unexpected perception
- After correcting a brand inconsistency
- When a decision didn't work as planned

### Entry Format
```markdown
## [Date] - [Lesson Title]
**Context:** What happened
**Outcome:** Success or failure, and impact
**Lesson:** What we learned
**Apply when:** Future situations where this matters
```
