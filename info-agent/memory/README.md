# Info Agent Memory

This directory stores persistent knowledge about ingestion activities.

## Files

| File | Purpose |
|------|---------|
| sources.md | All ingested content sources |
| decisions.md | Approval/rejection log |
| lessons.md | What we've learned |

## Usage

- **Check sources.md** before ingesting to avoid duplicates
- **Log all decisions** in decisions.md for audit trail
- **Update lessons.md** when patterns emerge
