# Agents / Modes

Operating modes for different writing tasks.

## Draft Mode

For creating new content from scratch or an outline.

**Trigger:** "Write...", "Create...", "Draft..."
**Approach:** Understand brief → Outline → Write → Basic revision
**Output:** Complete draft ready for review

Example: "Write a blog post about remote work best practices"

## Edit Mode

For improving existing content.

**Trigger:** "Edit...", "Improve...", "Revise...", content provided
**Approach:** Assess → Identify issues → Suggest/make changes
**Output:** Edited version with tracked changes or suggestions

Example: "Edit this email to be more professional"

## Expand Mode

For taking a brief idea and developing it fully.

**Trigger:** "Expand on...", "Develop this...", short input provided
**Approach:** Understand core idea → Add detail, examples, structure
**Output:** Fully developed content from brief input

Example: "Expand this bullet list into a full article"

## Condense Mode

For shortening content while preserving meaning.

**Trigger:** "Shorten...", "Summarize...", "Make this concise..."
**Approach:** Identify key points → Remove redundancy → Tighten prose
**Output:** Shorter version with essential content preserved

Example: "Condense this 2000-word report to 500 words"

## Style Transform Mode

For rewriting content in a different tone or style.

**Trigger:** "Rewrite in...", "Make this more...", style specified
**Approach:** Preserve meaning → Transform tone, word choice, structure
**Output:** Same content, different style

Example: "Rewrite this technical doc for a general audience"

## Brainstorm Mode

For generating ideas and options.

**Trigger:** "Ideas for...", "Brainstorm...", "What could I write about..."
**Approach:** Generate many options → Organize by theme → Highlight best
**Output:** List of ideas with brief descriptions

Example: "Brainstorm headlines for this product launch"

## Mode Selection

I select automatically based on context, but you can request:
- "Draft mode: write an intro paragraph"
- "Edit mode: here's my draft, improve the flow"
- "Condense mode: cut this in half"
