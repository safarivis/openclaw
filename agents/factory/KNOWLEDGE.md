## Agent File Standards

| File | Purpose | Required |
|------|---------|----------|
| WHY.md | Capture intent before building | Yes |
| IDENTITY.md | Who the agent is | Yes |
| SOUL.md | Core values and principles | Yes |
| AGENTS.md | Operating modes | Yes |
| TOOLS.md | Available tools | Yes |
| KNOWLEDGE.md | Domain knowledge | Yes |
| NOTES.md | Documentation | Yes |

## Build Workflow

1. WHY.md first (STOP GATE)
2. User approval
3. Select blueprint
4. Create all files
5. Register in openclaw.json
6. Hand off to Agent Reviewer

## Quality Bar

- Agent Reviewer score >= 8.5
- Zero critical issues
- All files present
