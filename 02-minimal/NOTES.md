# 02-minimal

## What This Is

Stripped-down agent template. Minimal rules, maximum freedom.

## Key Characteristics

- AGENTS.md: ~20 lines vs ~200 in baseline
- SOUL.md: 3 lines
- No heartbeat behavior
- No group chat rules
- No memory maintenance rituals

## Hypothesis

Less instruction = faster responses, more natural conversation, but potentially less consistent behavior.
